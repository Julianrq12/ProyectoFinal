Descripción del Proyecto

Este proyecto fue desarrollado utilizando tecnologías web modernas como HTML5, CSS3 y JavaScript. Además, se empleó la librería Leaflet.js para la visualización interactiva de mapas y OpenStreetMap como proveedor de mapas base. Estas herramientas permiten monitorear y visualizar geográficamente los centros informales de recolección de basura en La Florida, Engativá.
